## 🌊 What is Lead Wave?

**Lead Wave** is a powerful **WhatsApp Marketing Desktop Application** that helps businesses automate outreach, manage conversations, and run large-scale campaigns — all from a single Windows application.

This GitHub repository hosts the **complete sales/landing page website** for Lead Wave subscriptions. The website is:

- 🏗 Built with **pure HTML, CSS & JavaScript** — zero frameworks
- ⚡ **Single file deployment** — just one `index.html`
- 🎨 **Dark theme** with WhatsApp green branding (`#25D366`)
- 📱 **Fully responsive** — works on all screen sizes
- 🔍 **SEO optimized** — structured for Google indexing
- 💬 **WhatsApp-first** purchase flow — perfect for Indian buyers

> 🔗 **Live Website:** [https://getleadwave.in](https://getleadwave.in)

## 🚀 Quick Start — Run Locally

### Method 1: Just open in browser (simplest)

```bash
# Clone the repo
git clone https://github.com/yourusername/leadwave-website.git

# Go to folder
cd leadwave-website

# Double-click index.html OR:
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

### Method 2: Local dev server (recommended)

```bash
# Using Python (pre-installed on most systems)
python -m http.server 8080

# Using Node.js
npx serve .

# Using VS Code
# Install "Live Server" extension → Right click index.html → Open with Live Server

# Visit in browser:
http://localhost:8080
```

## 🌐 Deploy on GitHub Pages (Step-by-Step)

### 1️⃣ Create GitHub Repository

```bash
# Initialize local git repo
git init
git add .
git commit -m "🚀 feat: launch Lead Wave website"
git branch -M main
```

```bash
# Create repo on GitHub (via web) then:
git remote add origin https://github.com/YOUR_USERNAME/leadwave-website.git
git push -u origin main
```

### 2️⃣ Enable GitHub Pages

```
→ Open your repository on GitHub
→ Click "Settings" tab
→ Scroll to "Pages" in the left sidebar
→ Source: Deploy from a branch
→ Branch: main  |  Folder: / (root)
→ Click Save
→ Wait 2–3 minutes...
```

✅ Your site is now live at:
```
https://YOUR_USERNAME.github.io/leadwave-website/
```

```bash
# After making any changes:
git add .
git commit -m "✨ update: [describe what you changed]"
git push

# GitHub Pages auto-deploys within 1-2 minutes
```

## 🔗 Connect Custom Domain (getleadwave.in)

### Step 1: Create CNAME file

```bash
echo "getleadwave.in" > CNAME
git add CNAME
git commit -m "🌐 config: add custom domain"
git push
```

### Step 2: Add DNS Records in Domain Panel

Go to your domain registrar (GoDaddy / Namecheap / BigRock / any) and add:

```
Type    Host      Value                   TTL

A       @         185.199.111.153         3600

```
→ Settings → Pages
